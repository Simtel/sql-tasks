### **Отличия MyISAM и InnoDB**

| Описание      | MyISAM        | InnoDB       |
| ------------- | ------------- | -------------
| Транзакции  | Нет  | Да |
| Внешние ключи  | Нет | Да |
| Блокировки | Блокирует всю таблицу | Блокирует только строку |
| Одновременные запросы к разным участкам таблицы | Медленее | Быстрее |
